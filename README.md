# Vue 3 + TypeScript + Vite

## 环境

```
node ^14.18.0
```

## 安装依赖

```
npm install || yarn
```

## 启动服务

```
npm run dev
```

## 环境地址

```js
// 开发环境
   https://www.igancao.com/activity/dist/

// 生产环境
   https://www.igancao.com/activity/dist/  
```

## 全局状态管理Pinia

```js
$ npm install pinia

import { createPinia } from 'pinia'

app.use(createPinia())
```
